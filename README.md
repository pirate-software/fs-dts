# Ferret Software Twitch Extension Data Transformation Service (FS-DTS)

Periodically fetches data from the ferret software wiki to provide static content for the [Ferret Software Twitch Extension](github.com/pirate-software/fs-extension).

## Development

```bash
pnpm install
pnpm run dev
```

## Building

```bash
pnpm run build
```

## Running

```bash
pnpm run start
```
